A web app that lets you upload PDF documents and interact with them using AI

✨ Features
- 📝 **Summarize** — get a quick overview of any document
- 💬 **Chat** — ask questions and get answers from the document
- 🎴 **Question & Answer** — choose between 1 - 10 and get customs questions & answers from the document 

 🛠️ Technologies Used:
- 🐍 Python 
- 🌐 Streamlit
- 🤗 Hugging Face
- ⚡ Groq API
- 🔗 LangChain
- 🔍 FAISS
