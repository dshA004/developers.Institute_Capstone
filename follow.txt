Deployment Streamlit app steps:

We have our app in a virtual environment
To create a virtual environment, go to the project root folder and open a terminal.
Type in the command: 
python -m venv <virtual environment name>


Activate your virtual environment
./<virtual environment name>/Scripts/Activate

Install libraries

Create the requirements.txt file (while virtual environment is still activated)

Pip freeze > requirements.txt


Create a file called .gitignore

In this file add ./<virtualenvironment folder name>

Push your code app (the necessary files including the requirements.txt) to a public github repository that you created
